How to Compile and Run:
1. Open Command Prompt or VS Code terminal in this folder.
2. Compile all Java source files:
   javac *.java
3. Run the system:
   java Main

Main Functions:
- Staff login validation.
- Student registration form.
- Event category fee selection.
- Optional add-on activities.
- Payment method selection.
- Automatic total fee calculation.
- Registration record display in a table.
- Clear form, delete selected registration, and exit controls.

Business Perspective:
This system supports university or faculty event management by helping staff manage
student registrations faster, reduce manual fee calculation errors, and keep
participant records organized in one clean interface.
