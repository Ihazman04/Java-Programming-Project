# Student Event Registration and Fee Management System

## Overview
Java Swing GUI-based system developed for BIT33803 Java Programming Project.

## Features
- Login authentication
- Participant registration
- Event category selection
- Fee calculation
- Update and delete records
- Search/filter records
- Dashboard summary
- Registration statistics
- Payment summary
- CSV save/load

## Technologies
- Java
- Java Swing
- OOP
- ArrayList
- CSV Storage
